#### [Web Application User Guide](WEB_APP_USER_GUIDE.md)
End-user documentation for the Razor Pages web client.

**Topics Covered:**
- Getting started with the web interface
- UI overview and layout
- Creating, updating, and deleting tasks
- Registering progress with cumulative tracking
- Visual indicators (badges, progress bars, button states)
- Real-time features and auto-refresh
- Error handling and troubleshooting
- Complete task lifecycle examples
- FAQ and power user tips

**Best for:** End users and stakeholders using the web interface

**Topics Covered:**
- All 8 REST endpoints with full specifications
- Request/response examples in JSON
- cURL examples for each endpoint
- Data models and TypeScript interfaces
- Swagger/OpenAPI integration guide
- Business rules and validation
- CORS configuration
- Complete workflow examples
- Postman setup instructions
- Security notes and production recommendations

**Best for:** API consumers, frontend developers, and integration teams
